## API Requirements

```
  USED IN /api/pre.js
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `GEM_API_KEY` | `string` | **Required**. Your API key |

> [!NOTE]
> Add the variable to your environment variable.
